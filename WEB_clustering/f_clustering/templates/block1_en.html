Block 1

This web service is an implementation of sequential analysis of quantitative data defined in a finite feature space, which includes a preliminary step (intelligent setting of parameters) and THREE main steps:

  • clustering (fig. 1, 2, 3)

  • convex sets extraction (Fig. 4)

  • classification (Fig. 5), which uses the results of any of the two previous steps, by obtaining in the selected coordinate system "membership functions" of a data point (object/vector is the same here) from a test sample to a particular subset ("class") found on a trained sample.

The illustration of the one-dimensional case shows how based on the train sample, which was assigned each - to one of the two clusters "A" or "B", two membership functions are defined, the value of which depends on the density of concentration of objects from the corresponding cluster. After getting values of classificators on a test sample, we can measure the reliability of assigning points to the classes, which were created based on clusters.
