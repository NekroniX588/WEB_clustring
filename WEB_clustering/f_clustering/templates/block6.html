Как уже говорилось, результаты кластеризации/субкластеризации могут быть сохранены автоматически , на выбор:

А) в общем файле – набор точек будет идентичен набору во входном файле;

Б) в виде отдельных файлов (один файл = одно подмножество) для упрощения запуска дальнейшего цикла анализа, - то есть, - через загрузку результирующего файла с точками какого-либо выделенного подмножества, в качестве входного файла, для проведения дальнейшей кластеризации, уже отдельно внутри любого ранее выделенного подмножества данных.

Результаты классификации так же могут быть сохранены в виде файла (со значениями классифицирующих функций, рассчитанными для каждой точки данных в количестве , равном количеству найденных кластеров);

Примечание: значения данных, скачиваемых на любом этапе, после этапа подбора констант, выводятся и сохраняются в файлы результатов в Нормированном виде.
