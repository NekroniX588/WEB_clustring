Block 6

As already mentioned, clustering/sub-clustering results can be saved automatically, with a choice of:

A) in a shared file - the set of points will be identical to the set in the input file;

B) in the form of separate files (one file = one subset) to simplify the launch of a further analysis. In other words, we load the resulting file with the points of any selected subset, as an input file, for further clustering, already separately inside any previously selected subset of the data.

The classification results can also be saved as a file (with the values of the classifying functions calculated for each data point in each found cluster);

Note: the values of the data downloaded at any step, after the stage of selecting constants, are displayed and saved to the result files in the normalized form.

This web service provides access to a demo, simplified version of the complex of methods of multivariate statistical data analysis developed by us.
